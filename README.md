
# Basic Network Sniffer

## Description

A Python-based network sniffer that captures and analyzes network packets.

## Features

* Captures network packets
* Displays source and destination IP addresses
* Detects TCP, UDP, and ICMP protocols
* Shows packet payload information

## Technology Used

* Python
* Scapy

## Installation

pip install -r requirements.txt

## Run

python sniffer.py
